# Security Policy

## Supported Usage

This repository is intended for:
- educational cybersecurity research
- digital forensics demonstrations
- ethical steganography experiments
- authorized laboratory environments

## Responsible Usage

Users are responsible for ensuring compliance with:
- local laws
- cybersecurity regulations
- organizational policies
- authorized testing requirements

## Prohibited Usage

Do NOT use this software for:
- unauthorized system compromise
- malware distribution
- credential theft
- illegal surveillance
- unlawful data concealment

The project maintainer assumes no liability for misuse,
damage, or unlawful activities performed using this software.
